# 概述

[!UICONTROL Journey Orchestration] 是与Adobe Experience Platform集成的应用程序服务。 它允许您利用存储在事件或数据源中的情境数据构建实时业务流程使用案例。

* **[如何找到旅程编排帮助](/help/how-to-find-help-with-journey-orchestration.md)**   <br>
